English Email Writing Practice

Deze oefening toont na het inloggen één schrijfopdracht op A2-B1 niveau. De student schrijft een e-mail in het Engels, krijgt meteen feedback, en kan het resultaat daarna automatisch doorsturen.

Belangrijk voor e-mail doorsturen:
Deze pagina werkt direct op GitHub Pages voor de opdrachten en feedback. Het automatisch mailen naar jouw adres werkt alleen nadat je hieronder in de code een extern formulierendpoint koppelt, bijvoorbeeld Formspree of een Google Apps Script webhook. Zonder zo'n koppeling blijft alles lokaal in de browser.
